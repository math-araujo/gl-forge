# gl-forge

Personal C++ OpenGL 4.6 framework for graphics projects.

## Build

```
git clone https://github.com/math-araujo/gl-forge.git

cd gl-forge

# Run `cmake --list-presets` to see all available presets

# On Windows + MSVC
cmake --preset=msvc

cmake --build build --config Release

# Or on Linux
cmake --preset=unix

cmake --build build
```
